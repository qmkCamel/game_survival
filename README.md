# 互联网职场生存 - 文字冒险游戏

这是一个关于互联网职场生存的文字冒险游戏，玩家需要在虚构的"996科技"公司中面对各种职场挑战和人际关系，通过做出明智的选择来影响自己的职业发展。

## 游戏主题

游戏聚焦于当代互联网行业的职场生态，涉及以下关键元素：
- PUA（职场精神控制）
- PIP（绩效改进计划）
- 升职加薪
- 裁员与赔偿
- 试用期挑战
- 办公室政治
- 摘桃子（功劳被抢）
- 低垂的果实（高回报低成本的任务）
- 排兵布阵（团队与资源管理）
- 职场打法（生存策略）

## 游戏规则

1. **生存与发展**：玩家需要在互联网公司中生存，面对各种职场挑战和人际关系。
2. **决策影响**：玩家通过选择不同的行动来影响游戏的发展，每个选择都会带来不同的后果。
3. **资源收集**：玩家需要收集职场"技能"和"盟友"来帮助自己生存。
4. **平衡策略**：玩家需要平衡工作表现和心理健康两个关键指标。

## 游戏指标

- **工作表现**：反映你在公司的业绩表现和领导评价，低于20%可能导致被裁员。
- **心理健康**：反映你的精神状态和压力水平，低于20%可能导致职业倦怠。
- **技能**：帮助你应对各种职场情况的能力，如"办公室情报"、"沟通技巧"等。
- **盟友**：可以在关键时刻帮助你的人脉资源，如"张总"、"小林"等。

## 游戏结局

游戏有多种可能的结局，包括但不限于：
- **职场赢家**：成功晋升，获得良好的职业发展
- **职业倦怠**：心理健康崩溃，不得不离开公司
- **被裁员**：工作表现不佳，被公司优化
- **平衡大师**：找到工作与生活的平衡点
- **职场政治大师**：精通办公室政治，成为公司高管
- **技术专家**：凭借过硬的技术能力获得尊重和特殊待遇
- **创业者**：积累足够的经验后离开公司自立门户

## 游戏特色

- **丰富的职场场景**：游戏中包含了多种真实的职场情境，如项目抢功、资源争夺等
- **多样的人际关系**：建立自己的职场盟友网络，应对各种人际挑战
- **战略性选择**：每个选择都会影响多个指标，需要权衡利弊
- **多重结局**：根据你的选择和最终指标，获得不同的职业发展结局

## 游戏开发

本游戏使用纯HTML、CSS和JavaScript开发，无需任何额外的依赖。

### 文件结构
- `index.html` - 游戏的主界面
- `styles.css` - 游戏的样式表
- `game.js` - 游戏的主要逻辑
- 场景文件：
  - `scenes_intro.js` - 开始和培训场景
  - `scenes_training_responses.js` - 培训回应场景
  - `scenes_office_politics.js` - 办公室政治场景
  - `scenes_projects.js` - 项目管理和团队协作场景
  - `scenes_endings.js` - 结局场景
  - `scenes_index.js` - 场景整合文件

### 扩展游戏
如果你想添加更多场景或内容，可以：
1. 在相应的场景文件中添加新的场景对象
2. 确保每个场景都有正确的`text`和`choices`属性
3. 通过`nextScene`属性正确连接到其他场景

## 关于项目

这个游戏旨在以轻松幽默的方式反映当代互联网职场的真实状况，希望能给玩家带来思考和共鸣。游戏中的"摘桃子"、"低垂的果实"和"排兵布阵"等元素都来源于真实职场中的现象。
